🐛 Troubleshooting Tips

  • Can't reach the server from another device?
    • Check both devices are on the same WiFi network
    • Try: ping 192.168.0.202 from the other device
    • Disable Windows Firewall temporarily: Settings → Windows Defender Firewall → Turn off
  • Page loads but no data shows?
    • Open browser DevTools (F12) → Console tab
    • Check for CORS errors (red messages about blocked requests)
    • API URL should show: http://192.168.0.202:3002/api
  • Backend returns error?
    • Ensure MySQL is running in XAMPP
    • Check backend terminal for error messages
